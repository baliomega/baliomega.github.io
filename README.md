# Nurul Izwan Dahlan – Personal Website

This is the source code for my personal website and online resume, designed and developed by me, Nurul Izwan Dahlan.

## Overview
- **Purpose:** Showcase my resume, skills, professional experience, and selected projects in a modern, accessible, and visually engaging format.
- **Features:**
  - Clean, responsive layout for desktop, tablet, and mobile
  - Animated, colorful abstract background shapes
  - Accessible, readable typography using the Inter font
  - Contact information (LinkedIn & GitHub)
  - **Projects section** with card UI to showcase work (e.g., Chrome Extensions, web apps)
  - Project cards are responsive and support multiple projects per row
  - Only two main files: `index.html` and `style.css` for simplicity and performance

## Projects Showcase
- The site now includes a "Projects" section with a modern card layout.
- Example: MindShuffler Chrome Extension is featured, with a description, features, and a link to the Chrome Web Store.
- You can easily add more projects by duplicating a card in the HTML.

## How to View
Simply open `index.html` in any modern web browser. No build tools or dependencies required.

## Deployment
You can deploy this site using GitHub Pages:
1. Push the repository to your GitHub account as `<username>.github.io`.
2. Enable GitHub Pages in the repository settings (if not already enabled).
3. The site will be live at `https://<username>.github.io/`.
